# Pokedex

Clone este projeto no GitHub

Depois do fork, clone o repositório em sua máquina usando o:

```sh
git clone ...
```

Com o repositório clonado, instale as dependências:

```sh
npm install
```

Com as dependências instaladas, execute o projeto:

```sh
npm run dev
```
